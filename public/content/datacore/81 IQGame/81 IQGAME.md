# 81 IQGAME

Dual N-Back working memory training game.

## Instructions
The goal of each level is to remember the state of the game exactly N timesteps ago.

- **Visual**: 3x3 grid (8 squares). Press 'a' when the current square matches the square N steps ago.
- **Audio**: Spoken letters. Press ';' when the current letter matches the letter N steps ago.

Official Game: [dual-n-back.io](https://dual-n-back.io/)
